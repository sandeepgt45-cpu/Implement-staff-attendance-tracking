# \# Doctor Attendance Tracking System

# 

# \## Project Overview

# 

# The Doctor Attendance Tracking System is a Java-based application developed to manage and monitor the daily attendance of hospital doctors and staff members. The system records attendance status, calculates attendance percentage, tracks absences, and generates attendance reports efficiently using Object-Oriented Programming concepts in Java.

# 

# This project is useful for hospitals and healthcare organizations to maintain attendance records digitally and improve staff management.

# 

# \---

# 

# \# Features

# 

# \- Add doctor/staff details

# \- Record daily attendance

# \- Mark Present or Absent status

# \- Calculate attendance percentage

# \- Track total absences

# \- Display attendance reports

# \- Menu-driven console application

# \- Uses classes, constructors, methods, and arrays/collections

# 

# \---

# 

# \# Technologies Used

# 

# \- Java

# \- OOP Concepts

# \- Arrays / ArrayList

# \- Scanner Class

# \- Methods and Constructors

# \# System Functionalities

# 

# \## 1. Add Doctor Information

# 

# The system stores:

# \- Doctor ID

# \- Doctor Name

# \- Department

# 

# Example:

# ```text

# Doctor ID: 101

# Doctor Name: Dr. Smith

# Department: Cardiology

# ```

# 

# \---

# 

# \## 2. Record Attendance

# 

# Attendance can be marked as:

# \- Present

# \- Absent

# 

# Example:

# ```text

# Enter attendance status (P/A): P

# ```

# 

# \---

# 

# \## 3. Calculate Attendance Percentage

# 

# Formula:

# 

# ```text

# Attendance Percentage =

# (Total Present Days / Total Working Days) × 100

# ```

# 

# \---

# 

# \## 4. Track Absences

# 

# The system maintains:

# \- Total absent days

# \- Total present days

# 

# \---

# 

# \## 5. Generate Attendance Report

# 

# The application displays:

# \- Doctor details

# \- Attendance percentage

# \- Total absences

# \- Attendance status
